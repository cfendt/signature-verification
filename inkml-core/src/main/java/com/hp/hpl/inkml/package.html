This package provides the core InkML processor library. InkML processor is a necessary component in developing applications with InkML as digital ink format. The InkML processor takes the InkML document and schema and processes the information so that it may then be used by applications requesting the information.

It supports both ‘archival’ and ‘streaming’ style of ink processing.It provides a pluguable architecture to acheive parse method independent and parser independent implementation. The Current implementation of this library provides a DOM implementation using Apache Xerces XML parser that bundled in JDK 1.5 or higher.
